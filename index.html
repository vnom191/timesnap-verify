What visual screening means

This score is deliberately not called an “AI detector”.

From a browser-only page, it is not honest or reliable to claim that a photo is definitely AI-generated, edited, or unedited. The score combines readable-code confidence, registry match status, and an optional exact file-fingerprint comparison when your Firebase record stores a SHA-256 hash.

For forensic image analysis or a real AI-manipulation model, use a protected server-side endpoint or Firebase Cloud Function. Do not expose private AI API keys inside this HTML file.
